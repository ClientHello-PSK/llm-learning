> 目标：短期内快速集中学习，按知识点逐层推进。每层是完整知识块，可直接读完。

## 整体结构（4 层推进）

```
L1 基础理论         ─► 先建立 DP 的物理直觉（多副本、layout）
   │
L2 推理 DP 特化     ─► 区分推理场景与训练场景的本质差异
   │
L3 vLLM 实现详解    ─► 把抽象概念落到 vLLM 源码
   │
L4 进阶场景         ─► DP 与 EP/PP/TP 的组合、wide-EP 部署
```

> DP 不像 TP/PP 那样有硬件特化（Ascend vs CUDA 差异小），所以没有 L5 层。
